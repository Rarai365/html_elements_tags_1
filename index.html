Lets Learn about HTML elements/tags

What are HTML elements?

HTML elements are the building block of web pages that define the structure, context, and presentation of page.
They are created using HTML tags, which are used to wrap around a piece of content. It tells web browser to intrepret and structure a part of an HTML documents.
They can contain content, semantic meaning, and formatiing instructions.
